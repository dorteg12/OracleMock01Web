Introduction

Zones

DNS Zones hold the trusted DNS records that will reside on Oracle Cloud Infrastructure's nameservers. You can create public zones with publicly available domain names or private zones to define your own domain name for private address resolution.

The Oracle Cloud Infrastructure Domain Name System (DNS) service lets you create and manage your DNS zones . You can create zones, add records to zones, and allow Oracle Cloud Infrastructure's edge network to handle your domain's DNS queries.

Creating a Zone and Setting up a Domain

Estimated Time: 30 minutes

Objectives

In this Lab, you will:

  • Aquire a domain
  • Create a zone in Oracle
  • Add nameservers to your domain

Prerequisites

  • An Oracle Cloud Account - please view the Getting started page for instructions on getting an Oracle Cloud Account
  • Experience with domain configuration

Note: It is posssible to aquire a domain from any site. This lab will be getting a domain from name.com, but any other site should follow similar procedures.

Task 1: Aquire a Domain

Name.com is a domain site where you can purchase a domain. This lab will use that site to get a domain that will eventually help link both instances that have been created in lab 1. However in order to do that, there must first be domain and zone.

  1. Got to name.com and search for a domain.

  2. If the domain you searched for is available you must purchase it in order to configure the nameservers and manage DNS records.

    once you have your domain leave the site open and procede to the next task

Task 2: Create a Zone

  1. Click the Navigation Menu in the upper left. Navigate to Networking, and select Zones under the DNS Management section.

  2. Select create zone
    • Enter the domain name that was aquired by completeing task 1 for the Zone Name
    • Choose the compartment that you have been working off of
    • Select Primary as the zone type
    • Select create to start the process of creating the zone
  3. Once the Zone is active there will Nameservers displayed on the screen.

    These nameservers need to be configured to the domain

  4. Go back to the name.com page and select the my domains tab on the top right and select the domain aquired
  5. Under the Advanced Security section of the domain select Manage Nameservers and add the nameserver given by the zone.

    6. Note: It could take time for the configurations to be completed. Once that is completed, the Zone is finnished. However, changes on the domain side will still be needed and will be done in lab 3.
