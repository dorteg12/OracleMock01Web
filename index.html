Welcome to Oracle, this website demonstrates how to deploy a resilient website using 2 Compute instances, 2 Virtual Cloud Networks and use a bucket to synchronize media objects for the website.

Demo Lab

The Demo Lab page contains step by step instruction on how to set up a Resilient website in Oracle. This is done by using two compute instances, each running the same webserver. The failover is done with help of Geolocation Steering.
